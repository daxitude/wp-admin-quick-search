Adds an autocompleting search box to the admin bar for quickly finding and jumping to a post

== Description ==

The input currently searches in post_title only.

This plugin requires javascript.

== Installation ==

See [Installing Plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

1. Download the zip from here or from [Github](http://github.com/daxitude/wp-admin-quick-search) and drop it into your site's wp-content/plugins directory.
1. Navigate to your site's Admin->Plugins section (wp-admin/plugins.php) and activate the plugin.
1. Start typing a post title into the new input in the Admin Bar!
